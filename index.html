Rubik's Notation

The following notation will be used to refer to the different sides of the cube through out this article:

Front \(F\)
Down \(D\)
Left \(L\)
Right \(R\)
Up \(U\)
Back \(B\)

The same notation will refer to clockwise, \(90 ^\circ \) rotations of the corresponding face. For example, \(F\) will refer to a clockwise, \(90 ^\circ \) roation of the front face. To correspond with future group theory notation a counterclockwise rotation will be denoted by \(F'\). Additionally, a superscript added to the notation will indicate the number of times that action is performed. For example, \(FF\), a rotation of \(180 ^\circ\) of the front face, will be denoted \(F^{2}\).

Groups

What is a Group?

We will begin with the definition of a group. However, before we can define a group, we must define the concept of a binary operator.

Def Binary Operator:

Given two elements \(a,b\) of a set \(S\), a binary operator \(*\) maps these pair of elements to an element in that same set \(S\). A more rigorous definition can be found here.

For example:
Addition is a binary operation under the \(+\) operator.
\(2,3 \, \epsilon \, \mathbb{Z} \)
\( 2 + 3 = 5 \, \epsilon \, \mathbb{Z} \)

Def Group:

A Group is a set, let's call it G, paired with a binary operation \(*\) which fulfills the following axioms:

  1. The binary operation \(*\) must be closed.
    This means for any elements \( g, \,h \, \epsilon \, G\), \( g * h \) must also be in G.

  2. The binary operation \(*\) must be associative.
    This means for all \(a, b, c \, \epsilon \, G \):
    \( (a * b) * c = a * (b * c) \)

  3. There exist some element \(e\) in \(G\) such that for all \( x \, \epsilon \, G\):
    \( e * x = x * e = x\)

  4. For every element \(a \, \epsilon \, G \), there exists an element \(a' \, \epsilon \, G \) such that:
    \(a * a' = a' * a = e\)

    This \(a'\) is known as the inverse of \(a\). It also follows that \(a\) is the inverse of \(a'\). \(a'\) can also be denoted \(a^{-1}\).
It is worth noting that commutativity under the binary operation \(*\) is not a requirement. Commutative groups are a special type of group known as abelian groups.

Groups are sometimes denoted \(\langle G, * \rangle \).

Theorems

The following theorems are important to keep in mind when dealing with groups:

  • The identity element, \(e\), is unique
  • If \( a * b = e\), then \( a = b\,' \)
  • If \(a * x = b * x \, \), then \(\, a = b\)
  • The inverse of \((a*b)\) is \(b\,'*a'\)
  • \((a')' = a\)

Examples of Groups

Here are some examples of commonly known groups:

  • The set of integers, \( \mathbb{Z} \), form a group under addition \( + \).
    Addition is an associative operation.
    The identity element is \(0\) ( \(a + 0 = a\) ).
    The inverse of any \(a \, \epsilon \, \mathbb{Z}\) is \(-a\).
  • The set of nonzero rational numbers form a group under multiplication.
    \(1\) is the identity element.
    The inverse of any element \(x\) is \( \frac{1}{x} \).
  • The set of \(n \times n\) non-singular matrices form a group under multiplication.
    The identity is the \(n \times n\) identity matrix.
    The inverse of any element is its inverse matrix.

    This group is non-commutative or non-abelian. The Rubik group is also non-abelian.
