Groups

What is a Group?

We will begin with the definition of a group. However, before we can define a group, we must define the concept of a Binary Operator.

Def Binary Operator:

Given two elements a,b of a set S, a binary operator \(*\) maps these pair of elements to an element in that same set S. A more rigorous definition can be found here.

For example:
Addition is a binary operation under the \(+\) operator.
\(2,3 \, \epsilon \, \mathbb{Z} \)
\( 2 + 3 = 5 \, \epsilon \, \mathbb{Z} \)

Def Group

A Group is a set, let's call it G, paired with a binary operation \(*\) which fulfills the following axioms:

  1. The binary operation \(*\) must be closed.
    This means for any elements \( g, \,h \, \epsilon \, G\), \( g * h \) must also be in G.

  2. The binary operation \(*\) must be associative.
    This means for all \(a, b, c \, \epsilon \, G \):
    \( (a * b) * c = a * (b * c) \)

  3. There exist some element \(e\) in \(G\) such that for all \( x \, \epsilon \, G\):
    \( e * x = x * e = x\)

  4. For every element \(a \, \epsilon \, G \), there exists an element \(a' \, \epsilon \, G \) such that:
    \(a * a' = a' * a = e\)

    This \(a'\) is known as the inverse of \(a\). It also follows that \(a\) is the inverse of \(a'\).
It is worth noting that commutativity under the binary operation \(*\) is not a requirement.
